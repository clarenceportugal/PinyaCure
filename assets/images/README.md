# Images Folder

Place your pineapple parts illustration image here.

## File Name
Name your image file: `pineapple_parts.png`

## Supported Formats
- PNG (recommended)
- JPG/JPEG
- SVG (if supported)

## Usage
The image will be automatically displayed in the Home screen.

## Note
After adding the image file, run `flutter pub get` to ensure it's properly registered.
